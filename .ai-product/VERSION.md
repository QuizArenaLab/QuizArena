# Purpose

The Version document serves as the official record of the current product version.

It provides a single source of truth for the product's release status, version history, compatibility, lifecycle stage, and governance.

Unlike Release Notes, which describe what changed in a specific release, this document always reflects the current official version of QuizArena.

# Version Classification

| Type  | Description                                                                                 |
| ----- | ------------------------------------------------------------------------------------------- |
| Major | Significant product evolution introducing new capabilities or architectural changes.        |
| Minor | New features and enhancements that remain backward compatible.                              |
| Patch | Bug fixes, security improvements, and performance optimizations without functional changes. |

---

# Version Numbering Policy

QuizArena follows Semantic Versioning.

```
MAJOR.MINOR.PATCH
```

Example:

- v1.0.0 — First Public Release
- v1.0.1 — Bug Fix Release
- v1.1.0 — Feature Update
- v2.0.0 — Major Platform Evolution
